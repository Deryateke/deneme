Github'a veri gönderirken yazılan komutlar ve açıklamaları

echo "# deneme" >> README.md ==> readme.md dosyası oluşturur.

git init ==> locale proje ve repo arasında bağlantı kurar

git add . ==> ddosyaları ve yapılan değişiklikleri github'a ekler (. tüm dosyaları ekle anlamına gelir)

git commit -m "first commit" ==> açıklama kısmımız

git branch -M main ==> projenin gönderileceği github reposunun branch

git remote add origin https://github.com/{kullanıcıadı}/{projeadı}.git ==> yapılan değişiklikleri uzak sunucuya gönder

git push -u origin main ==> yapılan değişiklikleri ekle

Locale'de yapılan değişiklikleri github'a gönderme

1. Adım: git add . (değişiklik yapılan tüm dosyaları githuba ekle)

2. Adım: git commit -m "" (yapılan değişikliklerle alakalı not kısmı notlar "" içerisine yazılır)

3. Adım: git push (yapılan değişiklikleri github'a gönder)

Githubdaki bir projeyi cihazıma nasıl indiririm ?

github'daki repo sayfasında arkaplanı yeşil olan code yazılı butona tıklanır. Açılan pencerede "Clone using the web URL." yazının üstündeki https kodunu kopyalanır. Ardından terminal/git bash projeyi klonlamak istediğimiz konuma getiririz (ör: cd desktop). terminal "git clone {kopyalanan https url'i}" yazı lır ve enter a tıklanır. Bu yöntemle githubdan projeyi cihazımıza indirmiş olur ve bu yöntem bize github'a tekrardan push yapma imkanı sunar. Bu yöntemi kullanabilmek için repoya veri ekleme iznimizin olması gerekir.

İkinci yöntem dosyayı sadece indireceğimiz tekrardan push lama yapmayacağımız durumda download zip metotu ile indirebiliriz.
